Information about UMAX firewire/IEEE1394 scanners

I try to add support for UMAX firewire/IEEE1394 scanners like the UMAX Astra 6400 to the sane-umax scsi backend. In the moment firewire scanners do not work. After some changes in the linux sbp2-firewire kernel module I am able to turn the lamp of the scanner on and off and to start a scan (the scanhead already does move). But I am not able to get any image data from the scanner yet. It is unknown if this is a problem in the sane-umax backend or in the firewire kernel modules.

I can not promise that the backend will support firewire scanners in the future but I think there is a good chance to get it work.
Author: Oliver Rauch
